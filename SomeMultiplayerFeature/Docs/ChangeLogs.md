# 0.5.0

- 修复客户端玩家在自定义角色时，`ban`命令无效的问题
- 添加`ping`命令
- 添加`players`命令，查看在线的玩家和其所在的位置
- 更新`模组限制`功能，现在会要求该模组版本必须是最新版本。

- 重写`踢出延迟玩家`功能
